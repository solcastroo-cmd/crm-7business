WhatsApp Business

Resposta automática com IA GRÁTIS
✅ Webhook pronto

Use o Z-API — o jeito mais simples de conectar WhatsApp no Brasil. Sem Twilio, sem Meta, sem cartão.

  1. Acesse z-api.io e crie conta grátis (7 dias free)
  2. Crie uma instância → escaneie o QR Code com seu WhatsApp
  3. Em "Webhooks" cole a URL abaixo
  4. Pronto — toda mensagem será respondida pela IA!
http://localhost:3001/webhook/whatsapp

Instagram DM

Resposta automática no Direct META
⏳ Configuração necessária

Conecte sua conta profissional do Instagram para responder DMs automaticamente com IA.

  1. Acesse developers.facebook.com → Criar app
  2. Selecione Instagram → Messenger API
  3. Configure o Webhook com a URL abaixo
  4. Verify Token: 7business_ig_token
  5. Assine o evento messages
  6. Cole o Page Access Token no .env
http://localhost:3001/webhook/instagram
Verify Token: 7business_ig_token

Como funciona?
